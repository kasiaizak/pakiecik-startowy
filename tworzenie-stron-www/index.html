Budowa strony internetowej

Strony internetowe składają się z warstw. Najniższa z nich (ale w sumie najważniejsza), warstwa 0, to TREŚĆ. Kolejne to: warstwa 1 - semantyka za którą odpowiada kod HTML, warstwa 2 - prezentacja za którą odpowiadają CSS oraz warstwa 3 - interakcja, czyli JavaScript.
Wyższe warstwy mogą wpływać na wszystkie warstwy poniżej, czyli: JavaScript > CSS > HTML > TREŚĆ.

HTML - HyperText Markup Language

HTML opakowuje i opisuje TREŚĆ strony - mówi przeglądarce o strukturze dokumentu i o znaczeniu treści na stronie (semantyka). Treść to wszystko co jest nośnikiem informacji: teksty, nagłówki, linki, listy, definicje, obrazki, video, formularze, itd.

HTML posiada swoją zdefiniowaną składnię i nie jest językiem programowania - jest językiem znaczników. Pliki mają rozszerzenie: .html

Struktura HTML

Budowa znaczników:

<znacznik atrybut="wartość">treść</znacznik>

Jak się później przekonamy atrybuty nie zawsze są wymagane.

Szkielet strony internetowej:

Strona składa się z "głowy" <HEAD> i "ciała" <BODY> dokumentu. Warto wiedzieć, że znacznik <HEAD> nie jest wymagany - strona internetowa może istnieć bez niego, ale w praktyce każda strona go zawiera.

<HTML>
    <HEAD>
    </HEAD>

    <BODY>
    </BODY>
</HTML>

Znaczniki w <HEAD>

  • <link> - pozwala podlinkować m.in. plik z arkuszami styli - CSS
    <link href="style.css" rel="stylesheet">
  • <style> - służy do osadzenia reguł CSS bezpośrednio w dokumencie HTML - obecnie rzadko stosowane.

Znaczniki w <BODY>

  • <main> - główna część dokumentu
  • <section> - sekcja dokumentu, może zawierać kolejne sekcje - każda z nich powinna zawierać nagłówek h2-h6
  • <header> - nagłówek dokumentu / sekcji
  • <footer> - stopka dokumentu / sekcji
  • <h1-h6> - tekstowe nagłówki tytułowe dokumentu / sekcji (h1 jeden na całą stronę i powinien skrótowo, jak najlepiej opisać treść strony
  • <p> - paragraf

CSS - Cascading Style Sheets

Kaskadowe arkusze stylów. Odpowiadają za prezentację wizualną na stronie - wpływają na wygląd elementów HTML na ekranie, wydrukach itp.
CSS można umieszczać w sekcji <HEAD> w znacznikach <style>, ale czytelniejsze jest umieszczenie ich w oddzielnym pliku .css i importowanie w <HEAD> za pomocą znacznika <link>.

Składnia CSS to reguły

selektor {
    właściwość: wartość;
}
